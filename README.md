# FilePad - Website

This is the official website for FilePad, a macOS Finder right-click video compression app.

## Live Website

The website is hosted at: `https://filepad.app`

## About FilePad

FilePad is a macOS app that adds two Finder right-click actions: instant video compression and a lightweight FilePad strip for batch compression.

### Features

- Right-click in Finder → FilePad: Compress Video
- Right-click in Finder → FilePad: Add to FilePad (batch strip)
- Local compression with a balanced H.264 preset
- Progress window with Reveal in Finder
- Services menu customization via System Settings

## Website Structure

Built with [Astro](https://astro.build/).

- `src/pages/` - Page components
- `src/layouts/` - Layout templates
- `src/content/blog/` - Blog posts in Markdown

## Development

### Local Development

1. Clone this repository
2. Install dependencies: `npm install`
3. Start dev server: `npm run dev`

### Build

```bash
npm run build
```

The website is automatically deployed to GitHub Pages when changes are pushed to the main branch.
